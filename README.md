# DAX Depot - Power BI Advanced Analytics Project

## 📊 Project Overview

This project was developed in Power BI as part of the DAX Depot assignment. The objective was to build a backend data model and perform advanced analytical calculations using DAX. All analytical results are displayed through a Matrix Visual as required by the project guidelines.

## 📂 Dataset Tables

* Sales_Fact
* Returns_Fact
* Customer_Dim
* Product_Dim
* Date_Dim
* Region_Dim
